This package contains following String Helpers
1. RemoveSpace : Remove space from string.

2. RemoveLastCharacter : Remove last character from string.

3. RemoveFirstCharacter : Remove first character from string.

4. ToString : Create a string from an array of string using StringBuilder class and specified separator.

5. ReplaceParticularCharacter : Replace the old character with new character in string.

6. CreateMd5Hash : Create MD5 hash of a string.

7. ToSeoSlug : Convert the string into SEO friendly slug
